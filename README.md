# GIPHY

## How To Run

1. Make sure you local has installed Node >= 10.16 and npm >= 5.6
2. Create new file `.env` and copy `.env.example` into it
3. Run this command `npm install && npm start`

## Feature

1. Home
2. List of iron man gif
3. Search page
4. Pagination
